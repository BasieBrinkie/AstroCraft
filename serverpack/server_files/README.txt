You can start the server by running start.bat (windows) or start.sh (Unix)
The script will download additional needed files at launch such as minecraft.jar



### De/In-creasing RAM ###
If you open the start script with a text editor you will find a variable ARGS with a option "-Xmx 4G"

The 4G means 4 gigabytes of maximal usage. You can also use M to specify the amount of megabytes if necessary.
